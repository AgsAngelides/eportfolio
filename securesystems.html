Secure Systems Architecture


Introduction to the module

This module aims to introduce the concepts and principles of the Secure Systems Architecture (SSA) and it involves an introduction into basic architecture of modern operating systems, the relationship between it and the Distributed systems, secure system coding practices, the role of virtualisation in SSA.
We will also look into the current and future architecture challenges, limitations and opportunities.

Overview of the work completed

Unit overview

The aim of unit 1 is to introduce us to the processes, threads and schedulles as well as the distributed systems. We are also going to be learning how to set up and use OS containers (e.g. Dockers).

I was unable to install the Docker using the guidance provided by the university. The .io file appears to be an older Docker version so I used docker-ce instead which appeared to work better. The installation guidance provided by the uservisity did not work and got hung at one point.

After a while, debian based VM would not work, so I restarted my work, and installed a Ubuntu on my Virtual box and then used an article by Parveez (2022) to guide me through docker installation and testing on Ubuntu.

I then tried to follow the link to the page provided to learn how to use containers for Python code, however, I found the instructions very unclear. It was not clearly stated how exactly to build the dockerfile nor how to run it. All that prior knowledge was assumed to already exist by the recommended page Author.

After watching some more videos on youtube on how to set up a Docker file, I was able to create a docker file using the university provided link and run it.

References
Parveez, S., H. (2022) How to Install Docker on Ubuntu: A Step-By-Step Guide [Updated]. Available from: https://www.simplilearn.com/tutorials/docker-tutorial/how-to-install-docker-on-ubuntu [Accessed 10 August 2022].
